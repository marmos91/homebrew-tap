# marmos91/tap

Homebrew formulae for my projects.

## Usage

```bash
brew tap marmos91/tap
brew install marmos91/tap/<formula>
```

## Available Formulae

### Cubbit DS3 Drive

Sync local files with Cubbit DS3 cloud storage. Native macOS app with Finder integration via File Provider. [Repository](https://github.com/marmos91/cubbit-ds3-drive)

```bash
brew install --cask marmos91/tap/cubbit-ds3-drive
```

### DittoFS

A modular virtual filesystem with pluggable storage backends. [Repository](https://github.com/marmos91/dittofs)

| Formula | Description |
|---------|-------------|
| `dfs` | DittoFS server daemon |
| `dfsctl` | DittoFS CLI client for remote server management |

```bash
brew install marmos91/tap/dfs
brew install marmos91/tap/dfsctl
```
